Some Resources I find cool/useful

Software:

  • Operating System: PopOS 22.04 LTS. I find this works wonderfully for my needs. I use the standard Gnome with PopOS extensions, and a couple more as well.
  • Terminal Emulator: Wezterm. It's a nice terminal; I haven't experiemented with many more(I did try out Alacritty), and I find this nice and comfy.
  • Shell: zsh. I chose zsh over bash(because of modernity) and over fish(because of backward compatibility). It works well, and I do use some extensions such as zsh-syntax-highlighting and git-prompt.
  • Common GNU CoreUtils(and other CLI tools) and their alternatives: Some of these are just way better, hands down.
    • eza (ls)
    • fastfetch (neofetch)
    • bat (cat)
    • helix (vim/emacs)
    • btm and htop(both are super useful)
    • tldr (man); not an alternative - but useful nonetheless
    • typst (LaTeX)
  • Some other software that I find really useful, but not necessarily related to GNU, and in no particular order.
    • Wine - Wine Is Not an Emulator
    • Bottles - Nice software to properly use bottles(sandboxing and all)
    • Zen Browser - a modern browser based off of firefox
    • Bitwarden - password manager(maybe I should migrate?)
    • MuseScore - music notation software
    • Ableton - Music production software (NOT FOSS!!)
    • Ente - Google photos alternative
    • Arduino - to start out with basic electronics
    • Vivado - to program FPGAs (NOT FOSS!!)
    • iverilog - A verilog simulator
    • pandoc - universal document converter
    • ffmpeg - audio and video tool
    • yt-dlp - CLI audio/video downloader
    • rclone - rsync for cloud storage
    • asciinema - terminal recorder
    • binsider - A tool to analize ELF binaries
    • hexdump - display file contents in hex, decimal, octal or ascii
    • heh - a more colorful hexedit
    • WireShark - a network protocol analyzer
    • nchat - terminal based messaging frontend client
